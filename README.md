##用户手册
###1.应用程序功能的详细说明
一个游戏。可添加、选中并删除小球，删除时小球上的分值累加到得分；
可导入、导入游戏状态；可暂停游戏；退出时游戏记录可保存。
###2.应用程序运行环境的要求
Windows操作系统。已经采用静态编译，无需其他库。
###3.应用程序的安装与启动方法
双击exe即可运行。
